# İş Takip Otomasyonu 🚀

Bu proje, kurum içi personellerin, projelerin ve görevlerin uçtan uca yönetilebildiği kapsamlı bir İş Takip Otomasyonudur. Veri güvenliği ve sürdürülebilirlik göz önüne alınarak, tamamen Nesne Yönelimli Programlama (OOP) ve Katmanlı Mimari prensiplerine sadık kalınarak geliştirilmiştir.

## 💻 Kullanılan Teknolojiler ve Mimari
* **Backend:** C#, ASP.NET Core Web API
* **Frontend/UI:** Windows Forms (WinForms)
* **Veritabanı:** MS SQL Server
* **ORM:** Entity Framework Core
* **Mimari Yapı:** N-Layer Architecture (Entities, DataAccess, Business, WebAPI, UI)

## 🌟 Öne Çıkan Özellikler
* Güvenli kullanıcı giriş (Login) sistemi.
* Dashboard üzerinden anlık sistem istatistikleri (Toplam Görev, Personel vb.) takibi.
* Personel, Departman, Müşteri ve Proje (CRUD) yönetimi.
* Detaylı görev atama ve durum takibi.
* Web API üzerinden veritabanı ile iletişim.
